About This Website

Indian League Sports Hub is a simple educational project created using HTML and CSS. It provides basic information about the major sports leagues held in India such as:

  • IPL – Indian Premier League (Cricket)
  • ISL – Indian Super League (Football)
  • PKL – Pro Kabaddi League (Kabaddi)
  • Plus other sports leagues

Purpose

The main aim of this project is:

  • To showcase famous Indian sports leagues
  • To provide league overview, fixtures and recent highlights
  • To build a simple static multi-page website

Technologies Used

  • HTML (Structure)
  • CSS (Design)
